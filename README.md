# [DibsDash.com](https://dibsdash.com)
This repository contains all code for DibsDash's website, a minimalist utility app to help you and your friends coordinate food, drinks, and more for your next gathering. Think potluck sign-up sheets, but better!
### Coordinate party and potluck items with friends in less than a minute!
It's Simple:  
- Create an event
- Share link with others
- Add items
- Claim and modify items as needed
  
No more messy Google Sheets. List items easily. No accounts or downloads. 100% Free.
